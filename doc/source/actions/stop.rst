====
Stop
====

Synopsis
--------

**action name**: ``stop``

Stops a server instance

This action will allow you to stop a server instance on a compute host.

Configuration
-------------

Action parameters:

======================== ====== ======== ===================================
parameter                type   required description
======================== ====== ======== ===================================
``resource_id``          string yes      UUID of the server instance to stop
======================== ====== ======== ===================================
